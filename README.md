# Hornet Dashboard Mockup

This folder contains a draft mockup for the NABU–CorrelAid hornet dashboard.

## Purpose

The mockup is a discussion aid for reviewing:

* the dashboard’s main purpose and audience;
* the proposed layout and visualizations;
* possible MVP features;
* features that may belong to later versions;
* open data, hosting and maintenance decisions.

## Important

This is **not a final design or production dashboard**.

All displayed values are illustrative placeholders. The mockup does not contain real analysis results or confirmed scientific conclusions.

Observation counts must not be interpreted as animal abundance. Regional differences may also reflect reporting activity and data availability.

## Proposed main elements

* Germany-centred observation map
* Species and year filters
* Summary indicators
* Observations by federal state
* Observation trend over time
* Data-quality and interpretation notes
* Possible later research features

The final scope, data source, update process, hosting solution and public-display rules still require agreement with NABU.
